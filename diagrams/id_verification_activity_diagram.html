Diagram Description:

This activity diagram illustrates the ID verification process in the "What's It Like" application:

  • Multiple Entry Points: The verification can be initiated from either standard registration or after Google authentication
  • ID Upload: Users must upload an image of their Bosnian ID card
  • OCR Processing: The system uses optical character recognition to extract text from the ID image
  • Bosnian ID Focus: The system specifically processes Bosnian ID cards, extracting the ID number and name
  • Validation Checks:
    • ID format validation to ensure it matches Bosnian ID patterns
    • Duplicate checking to prevent multiple accounts with the same ID
  • Account Creation: Upon successful verification, the system creates the user account
  • Avatar Generation: The system generates an avatar using the user's initials from their verified name

The diagram highlights the mandatory ID verification step that ensures users are from Bosnia and Herzegovina, and the consistent avatar generation using the verified name information.
